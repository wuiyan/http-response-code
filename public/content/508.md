## 定义

```
HTTP 508 Loop Detected 表示服务器在处理请求时检测到无限循环，从而终止了整个操作。
```

## 描述

此状态码表示整个操作失败，因为它在处理带有 `Depth: infinity` 的请求时遇到了无限循环。 这通常在 WebDAV (Web 分布式创作和版本管理) 的上下文中给出。

当服务器作为网关或代理时，如果它在处理请求时检测到一个循环（例如，代理链中的请求循环），也可能会出现此错误。
