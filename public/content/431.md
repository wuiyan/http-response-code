# 定义

```
HTTP 431 Request Header Fields Too Large 表示服务器不愿意处理请求，因为请求头字段过大。
```

## 描述

HTTP 431 状态码表示单个请求头字段或所有请求头字段的总和过大，服务器拒绝处理该请求。

常见原因：
- 单个请求头字段值过长
- 请求头字段数量过多
- Cookie 数据过大
- URL 中包含过长的查询参数

服务器返回此状态码时通常不会关闭连接，允许客户端发送新的、头部较小的请求。这与 414 URI Too Long 不同，414 是针对整个 URI 过长。
