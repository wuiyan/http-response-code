## 定义

```
HTTP 423 Locked 表示请求的资源被锁定，无法访问。
```

## 描述

HTTP 423 是 WebDAV 协议中的状态码，表示请求的资源被临时锁定，客户端无法访问。

常见情况：
- 文件被其他用户或进程锁定编辑
- 资源正在被使用中，无法修改
- 管理员对资源进行了锁定操作

锁定机制用于防止多个用户同时修改同一资源导致冲突。通常会提供锁定信息，包括：
- 锁定所有者
- 锁定类型
- 锁定超时时间
