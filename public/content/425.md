## 定义

```
HTTP 425 Too Early 表示服务器不愿意处理请求，因为请求可能被重放。
```

## 描述

HTTP 425 状态码用于防止重放攻击，当服务器认为请求可能被恶意重放时返回。

使用场景：
- TLS 1.3 中的早期数据（0-RTT）可能被重放
- 服务器检测到潜在的重放攻击
- 客户端在不安全的连接上发送敏感请求

服务器通过拒绝处理可能被重放的请求来保护资源安全。客户端通常需要重新发送请求，但需要使用更安全的方式。
