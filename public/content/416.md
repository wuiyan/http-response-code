## 定义

```
HTTP 416 Range Not Satisfiable 表示客户端请求的范围无法满足，因为请求的范围超出了资源的实际大小。
```

## 描述

HTTP 416 常见于断点续传或分段下载场景，当客户端使用 Range 头请求资源的一部分，但指定的范围无效时返回。

出现情况包括：
- 请求的起始位置超出了资源大小
- 请求的结束位置超出了资源大小
- Range 头格式错误

服务器在响应中应该包含 Content-Range 头，指明资源的实际大小，格式为 Content-Range: bytes */<size>。
